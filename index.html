# 善善的第三本書：杜拜是瞥見阿拉伯世界的一扇窗

# 前言

我在2017年農曆春節期間，第一次和爸爸媽媽搭乘阿聯酋航空 (Emirates Airline) ，第一次直飛前往杜拜（Dubai）。當時我們選擇杜拜作為自助旅行的國家選項之一的原因是：

  1. 能夠直飛（隨著我長大了，而且有肢體障礙的關係，不再轉機才能讓跨國自助旅行仍然可能。）
  2. 飛行時間居中，約等於到夏威夷和澳洲所需的時間，所以可以飛出亞洲，但是又比起歐美還近。
  3. 杜拜是阿拉伯世界裡治安較好，對外較為開放，也持續不斷在開發中，適合觀光的一個城市。

我們第一次去杜拜的時候住的飯店，叫做杜拜羅夫飯店（Rove Hotel Dubai），它是一家四星級的飯店，位在杜拜市中心，公共設施非常齊全完整，交通便利，備有免費的無限網路，大型螢幕遊戲室，戶外游泳池，Starbucks，小型飯店超市，和24小時開放的健身房。直接從飯店步行到杜拜購物中心（Dubai Mall）需時十分鐘，但因為我使用輪椅的關係，直接步行前往杜拜購物中心（Dubai Mall）仍有些空間上的障礙，加上還要提購物袋，所以我們當時多數仍坐短程計程車來往杜拜購物中心。第一次看到需要大排長龍等計程車，且等待通道狹長不利輪椅通行時，購物中心有當地人主動熱心告訴我們，可以開另一道門，告訴計程車引導員我們家有人使用輪椅，則可以免排隊優先搭乘。

杜拜當地的假日是週五和周六，所以當地的飯店和餐廳，通常都會針對週五週六兩天，推出早午餐（brunch）。我們當時在杜拜羅夫飯店，就曾吃過令人印象十分深刻的早午餐，當日主餐是烤明蝦和奶油洋芋球，這道主菜料理令人口齒留香、念念不忘， 所以隔天我們又再次去飯店餐廳報到，只可惜每天的當日特餐都是隨時替換的，於是無法再度享用烤明蝦和奶油洋芋球這道美食!

我們今年第二次去杜拜住的飯店是五星級的杜拜皇宮飯店（Palace Downtown, Dubai），它就座落於杜拜購物中心旁，可正面直接看到世界第一高樓，169層樓高的哈里發塔（Burj Khalifah）， 杜拜皇宮飯店雖然富麗堂皇，入口處還有大片的藍色水池、棕欖樹及大型動物雕像，還有噴泉水池造景。 僅有三層樓高，所以搭乘電梯時，不必擔心常有其它樓層的住客共用電梯，在搭乘電梯上可說是寬敞自在，不擁擠稠密。

爸爸媽媽為了我看水舞方便，特地幫我訂了在房間就可以直接看到世界知名的杜拜大型水舞表演的水舞房（Fountain View Room）。 媽媽在行前就先告知這家飯店的服務人員，因為我的身體行動不方便，相對其他住客，會長時待在房間裡欣賞水舞，所以請盡可能安排水舞視野最好，且靠近電梯的水舞房。當我們抵達了位於二樓的255水舞房時，很開心地發現，飯店人員果然貼心地安排了距離電梯最近，且正對著水舞表演的水舞房。在沒有水舞表演的時段，也能看到大片的藍色水池，造景瀑布，世界第一高樓，169層樓高的哈里發塔，和樓下燈光絢麗奇幻的露天餐廳區及遊船經過。因為房間的視野極佳，即使長時待在房間裡，也絲毫不會感覺無聊。此外飯店真的有按照媽媽行前的請求，在我們出去的空檔才來打掃房間，無分清潔時段，這點讓我們感到十分方便。

值得一提的是，我們住在五星級的杜拜皇宮飯店的經驗讓我們再一次地體認到一分錢一分貨（服務）的道理，雖然我們並不崇尚奢華，但是杜拜皇宮飯店的水舞房，有著絕佳的視野和地理位置，讓我們居住的每一分鐘，時時有美景可直接從房內或陽台遠眺，確實完全有它一晚高單價住宿的價值。 房間的寢具一律使用最纖細柔滑的上等棉製品，甚至還能自選高級枕心，讓睡眠變成極致的享受！剛抵達時，房間內還有新鮮紅色玫瑰花瓣拼成愛心，和高雅的一塊慶祝生日的金箔巧克力蛋糕。有一天晚上，我們點了客房服務的晚餐來吃，這家飯店的客房服務人員，不只將晚餐送入房內，整台手推車平面就是一個完整的餐桌，上面鋪著精緻的桌巾，插上蘭花，所有餐點都搭配著典雅設計的餐盤和擺盤，以增加視覺享受，甚至一疊烤吐司都用緹花餐巾包覆以保溫，服務人員都非常溫文有禮，所以讓我們留下絕佳的印象！

媽媽說：「很多人對阿拉伯聯合大公國的杜拜的印象都停留在奢華的刻板印象上。其實，杜拜就像是一個成功而有遠見的國際商人，知道如何抓住全世界遊客的心，並努力地在持續發展和建設中，讓我們即使隔兩年再去，都會有全新的地標和值得參觀的新旅遊點可看可去。以欣賞和學習，取代羨慕或排斥奢華高級的事物，會大為提升自己的視野、 見地和振動頻率。同時，看過那麼多世界之最後，也會讓人變得更加謙卑涵容。」
