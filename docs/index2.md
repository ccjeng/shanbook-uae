# 前言二：媽媽的話

當我為了善善的第三本旅遊圖文書，回去瀏覽**2017年去杜拜拍的照片時，我才發現當時也拍了很多非常精彩的照片，但和2020拍的精彩照片精彩的成份完全是截然不同的**。

![](https://live.staticflickr.com/65535/49667926238_035d2daa85.jpg)

**可見得人生中的每一次經驗都不會是相同的，要珍惜每一次個別的人生經驗，珍惜每一個遇見過的人和經驗，它們彼此都不會和不曾是是複製品。每次總是很驚訝帶著這種重度多重障礙，全程使用輪椅，完全非生活自理，尚無口語，又機伶聰明，主見想法很多，快跟我一樣高的孩子到遠國從頭到尾全自助旅行，居然還能拍出滿坑滿谷的好照片的韌性、耐性和意志力驚人的自己。每次總不知道我們那一次又是怎麼辦到和達成任務的?**

![](https://live.staticflickr.com/65535/49667926218_c8836fe6d9.jpg)

和善善在週間課後寫杜拜的書時，**他往往查資料時，整晚坐得直挺挺，聚精會神，一直開心到大笑，很有成就感！就像寫論文一樣，每首水舞曲目都還對照歌名一一上YouTube聽歌詞，寓教於樂，效果更加深刻！**

有時看著每次遠行踏遍的照片和已走過的無數足跡，連我都會很驚訝，我不過隻身而來，也不過一個人離開兒時的家自己在台北生活，是如何變出顯化這所有所有一切的奇蹟的，就連單單三張飛歐美的機票都已經是很多人好幾個月的薪水了，我至今已不知顯化出多少張遠國機票！遑論真的不是任何人都能排得到假期，或是人生中罕見真正的自由。我剛出社會時日夜都在工作還自己一個月花一萬五不含水電租房，當時就為了半年就出遠國(埃及、義大利、澳洲那麼遠的)一趟！

真心地感恩我今生得以閱歷體驗過最多元廣度極致經驗的一切的一切的一切。人生永遠關乎自己最核心的體驗、觀點、學習和成長，一切其他都是參照物和配角，人生的藉假修真，在任何處遇中我決定是誰，人生才是真正的學校！

善善媽咪 賴素苑
