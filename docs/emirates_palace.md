# 阿布達比酋長皇宮飯店 Emirates Palace

阿布達比酋長皇宮飯店（Emirates Palace）是位在阿拉伯聯合大公國（UAE）的首都，也是第二大城阿布達比的**豪華八星級飯店，建築風格帶有濃厚的阿拉伯民族色彩。阿布達比酋長皇宮飯店這間富麗堂皇的海濱飯店，位於占地85公頃的景觀花園中**，它距離阿布達比海濱購物中心（YAS Mall）和阿拉伯聯合大公國首都阿布達比的歷史地標，也是最古老的石頭建築:阿布達比豪森宫殿（Qasr al-Hosn），皆為4公里。

**阿布達比酋長皇宮飯店華麗的客房皆設有陽台，可眺望景觀花園、游泳池或阿拉伯灣的景觀。共附設了14間餐廳和酒吧，包括了設在海灘的浪漫燒烤餐廳。** 阿布達比酋長皇宮飯店的其他設施包括了冒險泳池、休閒泳池、兒童遊樂場、慢跑道、自行車道、SPA中心，和2間健身中心，飯店內並設有私人海灘及碼頭，且提供自行車租借服務、瑜伽課程與駱駝騎乘體驗。

阿布達比酋長皇宮飯店是電影玩命關頭7（Furious 7）在阿布達比的拍攝場景之一，飯店前方如阿拉伯聯合大公國（UAE）的各大知名飯店，前有噴泉造景。**依照我們實地的觀察，在阿拉伯聯合大公國（UAE）這樣位處沙漠的國家，因為缺乏水源，所以區隔一個景點或是飯店的高級、豪華、受重視程度就是:看看外面有沒有水舞、水池、噴泉或植物花園造景。越是水池、水舞、噴泉，和種樹種花等花園植物造景越大面積的地方，就代表著這個景點或飯店是投入大量心力和成本去維持的高級豪華地區**，例如某些高速公路的路段，沒有刻意人工去造景，用心及耗費大量成本去維持的地方，則當然就是黃土一片的景觀。代表**物以稀為貴，越是缺水和植物的地區，相形之下，水池、水舞、噴泉類造景和植物花園造景才更顯珍稀難得!所以我們之後會再寫到，一年只開放幾個月的奇蹟花園，才會取名叫做奇蹟花園（Miracle Garden）！**

八星級的阿布達比酋長皇宮飯店和七星級的帆船飯店（Burj Al Arab）卓越的服務，都致力於讓每個顧客能感受到國王般的待遇與享受。**阿布達比酋長皇宮飯店有一條僅供各國元首和王室成員使用的專屬車道**，是不准其他人通行的，能通往專屬電梯，再直達貴賓的總統套房，襯托出他們尊貴不凡的身分地位。但是，**其實奢華感或是阿拉伯聯合大公國王室不是我們身心障礙者家庭想談的重點。我們一向比較想著墨於如何克服萬難，隨時保持彈性思維和應變能力，而仍然保有世界多元體驗的各種機會與可能！**

八星級的阿布達比酋長皇宮飯店和七星級的帆船飯店一樣，只開放給在裡面住宿或是預約用餐的客人才能進入，一般遊客若非預訂住宿用餐，是不能進入參觀的！所以我們就像進入杜拜的帆船飯店時一樣，預訂了下午茶，才得其門而入，得以進入參觀、體驗和用餐。**當天我們剛好有在阿布達比酋長皇宮飯店的一樓大廳，看到許多穿著典型白色傳統阿拉伯罩袍的當地阿拉伯人，在表演中東民族舞蹈：棍子舞（Al Ayala Stick Dance），通常稱為約瓦拉（Yowalah）的Al Ayyala舞蹈，是阿拉伯聯合大公國（UAE）的傳統集體舞蹈，該舞蹈還伴有傳統音樂: 皮革風笛、長笛和鼓是舞蹈中演奏的傳統樂器。約瓦拉在音樂和舞蹈方面都與眾不同，有濃濃的當地中東文化氣息**，我在家裡有時也會拿著鼓棒，看著當地棍子舞的影片，跟著節奏隨著點頭和揮動鼓棒，學習他們跳棍子舞。

談談我們到八星級的阿布達比酋長皇宮飯店喝下午茶的經驗，就是**整體的室內建築裝潢上，就連廁所內外，特別是在黃金圓柱上，處處使用了大量的黃金，放眼望去可見一片金碧輝煌，使用了大量的高級鮮花來插大型盆花。我們在阿拉伯聯合大公國（UAE）的自助旅行經驗中發現，區隔飯店或餐廳高級的程度，就在於鮮花使用的多寡和數量上。** 例如，在我們2020年住的五星級的杜拜皇宮飯店（Palace Downtown, Dubai）一樓大廳的圓型水池中，固定灑滿了新鮮紅色玫瑰花花瓣，針對慶祝生日或蜜月的住客，也會用大量的新鮮紅色玫瑰花花瓣鋪在高級彈簧床上，做成紅心愛心，和鋪排成祝福語。杜拜皇宮飯店餐廳的客房服務晚餐，也附上了一大朵高級的蘭花插花。可是，我們第一次去杜拜的時候住的四星級杜拜羅夫飯店（Rove Hotel Dubai），飯店或餐廳內設計就更為現代而簡潔，沒有放置任何鮮花。

**八星級的阿布達比酋長皇宮飯店的下午茶約為七星級的帆船飯店花費的一半，約兩三千圓台幣一客，所以甜點道數較少一些，帆船飯店的下午茶則多達五到七道三層甜點盤的份量。** 可是，當時可能是因為旅途疲憊，我除了喝了我的現打芒果汁之外，居然對阿布達比酋長皇宮飯店裡面的甜點興趣不大，反而猛吃銀盤裡提供放入茶中使用的白色和金黃色糖球，媽媽因此為我感到非常可惜，我怎麼不多品嚐一點這難得來一次的八星級飯店提供的其他高級的甜點。**阿布達比酋長皇宮飯店內的咖啡，可以選擇在咖啡表層上有做出酋長皇宮飯店建築造型的拉花圖案，再灑上金箔的飯店代表性飲品。**

**人生中，體驗過最高等級的服務、裝潢設計、食物，和多元而寬廣的世界文化經驗，是非常值得我們發自內心、最深深感恩的一件事!**
